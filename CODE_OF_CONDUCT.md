# Codigo de conducta

## Compromiso

En este proyecto nos comprometemos a ofrecer un entorno libre de acoso para
todo el mundo, independientemente de edad, dimension corporal, discapacidad,
etnia, identidad y expresion de genero, nivel de experiencia, nacionalidad,
apariencia, raza, religion u orientacion sexual.

## Ejemplos de comportamiento aceptable

- Usar un lenguaje acogedor e inclusivo.
- Respetar puntos de vista y experiencias diferentes.
- Aceptar criticas constructivas con elegancia.
- Centrarse en lo mejor para la comunidad.
- Mostrar empatia con otros miembros.

## Ejemplos de comportamiento inaceptable

- Lenguaje o imagenes sexualizadas y atencion sexual no deseada.
- Trolling, comentarios insultantes o despectivos, ataques personales o
  politicos.
- Acoso publico o privado.
- Publicar informacion privada de terceros (direccion, email, etc.) sin
  permiso explicito.
- Cualquier conducta que razonablemente se considere inapropiada en un
  entorno profesional.

## Aplicacion

El mantenedor del proyecto es responsable de aclarar las normas y de tomar
medidas correctivas justas y apropiadas en respuesta a cualquier
comportamiento inaceptable.

Se pueden reportar incidentes contactando al mantenedor a traves de los
issues del repositorio o, si requiere privacidad, mediante GitHub Security
Advisories.

## Atribucion

Este Codigo de Conducta es una adaptacion del
[Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
